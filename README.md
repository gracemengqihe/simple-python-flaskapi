# simple-python-flaskapi
This is a simple python flask api. 

## Create python virtual env
`python3 -m venv .venv`   

## Activate venv
`source <venv>/bin/activate`

## install flask
`pip3 install flask`
`pip3 install flask-sqlalchemy`

## Start up 
`export FLASK_APP=application.py`   
`export FLASK_ENV=development`   
`flask run` 

## Add data 
`python3 db.py`
